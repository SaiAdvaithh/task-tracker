# 🚀 Daily Task Tracker

A full-stack productivity tracking web app built using FastAPI and React.

## 🔥 Features

* User login system
* Add and manage daily tasks
* Status tracking (Done / Pending / Not Done)
* Comments for each task
* Date-wise tracking
* Progress bar visualization
* Task grouping (Completed / Pending / Not Done)
* Weekly analytics chart

## 🛠️ Tech Stack

* Backend: FastAPI (Python)
* Frontend: React + Tailwind CSS
* Database: SQLite
* Charts: Recharts

## 📊 Functionality

* Track tasks daily
* Monitor progress with visual indicators
* View weekly performance trends
* Organize tasks by status

## 🚀 Future Enhancements
